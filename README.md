# poetry-explorer-nextjs
A web app that utilizes Next.js and Tailwind CSS along with PoetryDB to provide an elegant way for users to search for and read poetry.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

FEATURES:
- Search for poems by Title or Author
- Dynamic Theme switching (3 themes that change colors, fonts, and images)
- Favorites page
- History Page
